# cotgen

- generate_from_prompts.py: 这是从https://github.com/djghosh13/geneval/blob/main/generation/diffusers_generate.py改过来的，生成geneval原本的prompts
- generate_from_cot.py: 从qwen生成的复杂prompt当中进行图像生成
- cot_to_sana.py: 把geneval的简单的prompt转化为复杂的cot prompt和可组合的prompt
- cot_outputs: 用cot_to_sana.py生成的geneval的复杂prompts
